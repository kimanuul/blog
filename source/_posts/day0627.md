## list comprehension (반복문을 한줄로 표시)


```python
"a" in "apple"
```




    True




```python
fruits = ['apple', 'kiwi', 'mango']
newlists = []

# 알파벳 a가 있는 과일만 추출 후, 새로운 리스트에 담기
for fruit in fruits:
  print(fruit)
  if "a" in fruit:
    newlists.append(fruit)
    print(newlists)
```

    apple
    ['apple']
    kiwi
    mango
    ['apple', 'mango']
    


```python
newlist = [fruit for fruit in fruits if "a" in fruit]
print(newlist)
```

    ['apple', 'mango']
